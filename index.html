Automated Web Application Deployment

CI/CD Pipeline using Docker, GitHub & GitHub Actions

Project Overview

This project demonstrates a complete DevOps workflow where source code is version-controlled with Git, managed on GitHub, containerized using Docker, and automatically built using GitHub Actions.

Technology Stack

  • Docker
  • Git & GitHub
  • GitHub Actions (CI)
  • HTML & CSS
  • AWS EC2 (Deployment Ready)

Workflow

Developer → GitHub → GitHub Actions → Docker Build → Deployment
